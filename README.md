# College Academic Data Analysis Project

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a dataset related to colleges, students, events, projects, and research activities. The goal of the project is to analyze academic data and extract useful insights using Python data analysis libraries.

The project demonstrates how data analytics can be used to understand academic patterns, student participation, research trends, and project popularity.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset Description

The project uses multiple datasets:

**students.csv**
Contains information about students including department, year, and other academic details.

**colleges.csv**
Includes details about colleges such as college name and state.

**events.csv**
Stores event-related information including event types and participation.

**projects.csv**
Contains student project data and project popularity based on likes.

**research.csv**
Includes research field data and research activities.

## Project Workflow

1. Import required Python libraries.
2. Load datasets using Pandas.
3. Perform initial data exploration.
4. Check for missing values.
5. Clean the data if required.
6. Perform exploratory data analysis.
7. Create visualizations to understand patterns.
8. Generate insights from the data.

## Visualizations Included

The project includes multiple visualizations such as:

* Department wise student distribution
* State wise college distribution
* Event type analysis
* Project likes distribution
* Research field analysis

These visualizations help in understanding trends and patterns in academic data.

## Key Insights

* Some departments have significantly higher student enrollment.
* Certain states contain a higher number of colleges.
* Workshops and seminars are common academic events.
* Some projects receive high engagement from students.
* Research fields such as Artificial Intelligence and Data Science show strong interest.
